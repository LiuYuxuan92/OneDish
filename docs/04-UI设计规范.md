# 简家厨 - UI设计规范

> 版本: 2.0
> 适用平台: React Native Web (跨平台移动端应用)

---

## 设计原则

### 设计理念

**温馨活泼 + 现代简约**

为有婴幼儿（1.5岁左右）的家庭提供温暖、亲切、易用的餐食规划体验。

### 设计目标

1. **温暖感** - 使用暖色调（橙色、米色）营造家的感觉
2. **易用性** - 清晰的视觉层级，大触摸目标（最小44px）
3. **信任感** - 一致的设计语言，专业但不过于严肃
4. **食欲感** - 颜色搭配激发对健康食物的兴趣

## 组件规范

### 按钮 (Button)

#### 主按钮
- **背景**: `primary.500` (#FF8C42)
- **文字**: 白色
- **圆角**: 12px
- **高度**: 48px (lg)
- **悬停**: 背景变为 `primary.400`
- **按下**: 背景变为 `primary.600`

#### 次按钮
- **背景**: 白色
- **边框**: 1px solid `primary.500`
- **文字**: `primary.500`

### 卡片 (Card)

- **背景**: 白色
- **圆角**: 12px (md) 或 16px (lg)
- **阴影**: 轻微上浮效果
- **内边距**: 16px (md) 或 24px (lg)

### 输入框 (Input)

- **背景**: `neutral.gray50`
- **边框**: 1px solid `neutral.gray200`
- **圆角**: 8px
- **高度**: 48px (lg)
- **聚焦**: 边框变为 `primary.500`

## 动画交互

### 动画持续时间

| 类型 | 时长 | 用途 |
|------|------|------|
| Fast | 150ms | 微交互（按钮按下） |
| Normal | 250ms | 标准过渡 |
| Slow | 350ms | 复杂动画 |

### 交互反馈

- **按钮悬停**: 上浮 1px + 阴影加深
- **卡片悬停**: 上浮 2px + 阴影加深
- **列表项按下**: 背景变暗

---

## 无障碍设计

### 触摸目标

- 最小触摸目标: **44×44px**
- 推荐触摸目标: **48×48px**
- 相邻触摸目标间距: **8px**

### 颜色对比度

- 正文文字: **4.5:1** 最低
- 大文字 (18px+): **3:1** 最低
- 用户界面组件: **3:1** 最低
