# OneDish 分享与会员功能设计

---

## 一、分享功能增强

### 1.1 分享海报生成
- 用户点击"分享"按钮
- 生成带小程序码的海报图片
- 保存到相册 or 直接分享

### 1.2 分享到朋友圈
- 支持 onShareTimeline
- 标题 + 封面图

### 1.3 分享卡片样式优化
- 更精美的分享卡片
- 显示"一鱼两吃"标签

## 二、会员功能设计

### 2.1 会员权益

| 功能 | 免费版 | 会员版 |
|------|-------|-------|
| 每日推荐 | 3 条 | 无限 |
| 菜谱收藏 | 20 个 | 无限 |
| AI 智能推荐 | ❌ | ✅ |
| 专属菜谱 | ❌ | ✅ |
| 广告展示 | 有 | ❌ |
| 价格 | ¥0 | ¥9.9/月 |

### 2.2 会员页面
- 展示会员状态
- 开通/续费按钮
- 会员专属菜谱入口

### 2.3 会员状态存储
- 本地存储会员状态
- 到期时间判断
